A full list of major IRP codes can be found here: Click Me Communication between usermode and filter drivers is possible if the driver calls FltCreateCommunicationPort and specifies a MessageNotifyCallback argument. A user mode program can use FilterConnectCommunicationPort and then FilterSendMessage. The buffers sent from FilterSendMessage are pre-filtered with ProbeForRead and ProbeForWrite (read for input and write for output) but may be unaligned and are vulnerable to race condition attacks because they reside directly in user mode address space.
