About:
IOBit Advanced Malware Fighter (version 6.2, and possibly lower) contains IMF Camera Protect.sys which is vulnerable to a stack overflow attack when IOCTL 0x8018E000 is sent via DeviceIoControl with a user defined size. Return addresses in the stack can be overwritten and allow for code execution or a DoS.
Security Risk Level:
CVSS (v3.0) = 7.2 High (personal estimation)
Possible Solution(s) / Fixe(s):
Use a fixed size for memory movement functions.
